Football Analytics

Modeling Defensive Compactness and Breakdowns in the Italian Men's National Team

In progress — a project focused on analysing defensive organisation, compactness, and tactical breakdowns using tracking and event data.

Overview

This project aims to study how defensive compactness changes across match phases, how spatial gaps emerge, and how tactical breakdowns can be identified using football tracking and event data.

Planned Methodology

  • Define compactness metrics based on team shape and inter-player distances.
  • Identify defensive line breaks and spatial disruption events.
  • Visualize defensive structures and tactical breakdowns over time.
  • Translate findings into clear tactical insights.

Technical Stack

Python, pandas, NumPy, matplotlib, Plotly, mplsoccer, tracking data, event data.
